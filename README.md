# introduncción

Con esta libreria, imprimir será mucho más facil y customizado. 

# instalación
Para descargar la libreria utiliza alguno de los siguientes comandos en la terminal:
- pip install easyprint
- pip3 install easyprint 

# Uso
La única función de easyprint es write. En write tienes que indicar el conjunto de elemento que imprimira el programa separandolos con comas, dentro de unos paréntesis. Un ejemplo seria:

```bash
import easyprint 

write(7, "Python3", 3.14)
```

La salida del ejemplo es:
```
7
Python3
3.14
```
